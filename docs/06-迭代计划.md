# 迭代计划

版本：v0.9
更新日期：2026-07-08

## V0.3 已交付

- 前端首页工作台原型。
- 官网和对标网站在方案、README、页面中明确列出。
- Spring Boot / MyBatis 后端工程骨架。
- 注册、登录、当前活动和公开作品列表基础接口。
- 数据库设计。
- OpenAPI 接口设计。
- 业务逻辑判断。
- 文件夹系统说明。
- 研发 PDF。

## V0.4 后台管理规划

1. `/root/` 后台入口。
2. root 初始化、创始人邮箱绑定、root 密码设置。
3. 用户表和管理员账号管理。
4. 活动配置、报名窗口、提交窗口和赛道配置。
5. 报名问卷字段配置。
6. 报名列表、详情、AI 初筛、人工审核。
7. 个性化邮件草稿、编辑和发送记录。
8. 项目看板、附件查看、展示状态管理。
9. 前台作品按赛道、票数、最新排序展示。
10. 登录用户点赞作品。

## V0.5 UI 信息架构

1. 梳理后台管理 UI：`/root/` Shell、概览、用户、活动、报名、项目、邮件、飞书、日志。
2. 梳理整体赛事预览 UI：赛事状态、身份状态、报名入口、提交要求、作品展示、点赞排序。
3. 明确后台和前台预览的关系：后台配置活动，前台读取配置，后台可按身份预览前台效果。
4. 输出后台管理和赛事预览的 ASCII 原型、组件清单、落地顺序和验收标准。

## V0.6 静态 MVP

1. 已新增 `/root/` 后台管理静态原型。
2. 已跑通 root 初始化、报名详情、AI 初筛、审核邮件、问卷新增、项目公开和赛事预览身份切换。
3. 已补充 Playwright 后台交互测试。

## V0.7 前台链路与问卷编辑 MVP

1. 首页已调整为活动精神和可报名黑客松活动入口。
2. 已新增单场活动页：活动状态、赛道、报名表单、提交要求和作品平台入口。
3. 已新增单场作品平台：赛道筛选、精选 / 票数 / 最新排序、mock 点赞和附件标签。
4. 后台活动管理里的报名问题已支持标题、类型、必填、占位提示和选项编辑。
5. 已更新自动化测试覆盖前台三层链路和后台问卷编辑。

## V0.8 实训开发优先级

1. 实现 root / admin 权限拦截器。
2. 实现 `/api/root/*` 初始化与登录接口。
3. 实现 `/api/admin/users` 用户表。
4. 实现活动、赛道、表单字段 CRUD。
5. 实现报名提交与后台审核。
6. 实现 AI 初筛 mock。
7. 实现邮件草稿 mock 和人工确认发送记录。
8. 实现项目提交、附件上传和视频禁用。
9. 实现后台项目看板和公开作品展示。
10. 实现点赞去重。

## V0.9 前后台动态联动 MVP

1. 已新增后台活动配置聚合接口：`GET /api/admin/events/current/config`、`GET /api/admin/events/{eventId}/config`、`PUT /api/admin/events/{eventId}/config`。
2. 已新增前台活动动态读取接口：`GET /api/public/events/{eventId}`、`GET /api/public/events/{eventId}/tracks`、`GET /api/public/events/{eventId}/form-fields`。
3. 后台活动管理页已支持编辑活动基础信息、报名窗口、赛道、报名问题、必填状态和选项。
4. 前台活动详情页已优先读取后端活动配置，失败时保留 mock fallback。
5. 已通过 Vite proxy 验证 `http://localhost:5173/api/public/events/1 -> http://localhost:8080 -> MySQL`。
6. 已给云端数据库活动 ID 1 写入 MVP 赛道和报名字段，前台活动页可动态展示。
