## V0.4 后台管理规划

1. `/root/` 后台入口。
2. root 初始化、创始人邮箱绑定、root 密码设置。
3. 用户表和管理员账号管理。
4. 活动配置、报名窗口、提交窗口和赛道配置。
5. 报名问卷字段配置。
6. 报名列表、详情、AI 初筛、人工审核。
7. 个性化邮件草稿、编辑和发送记录。
8. 项目看板、附件查看、展示状态管理。
9. 前台作品按赛道、票数、最新排序展示。
10. 登录用户点赞作品。

## V0.5 实训开发优先级

1. 实现 root / admin 权限拦截器。
2. 实现 `/api/root/*` 初始化与登录接口。
3. 实现 `/api/admin/users` 用户表。
4. 实现活动、赛道、表单字段 CRUD。
5. 实现报名提交与后台审核。
6. 实现 AI 初筛 mock。
7. 实现邮件草稿 mock 和人工确认发送记录。
8. 实现项目提交、附件上传和视频禁用。
9. 实现后台项目看板和公开作品展示。
10. 实现点赞去重。
