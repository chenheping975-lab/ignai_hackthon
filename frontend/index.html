Core workflow

核心功能拆解

报名

个人 / 团队报名

动态报名字段、队员信息、赛道选择、重复报名校验、AI 标签与摘要。

审核

审核与邮件 Agent

管理员最终决策，AI 生成邮件草稿，发送前可编辑、预览和定制化发送。

提交

项目资料提交

队伍、成员、项目说明、Demo、代码链接、PPT/PDF/HTML/ZIP/MP3/图片附件。

评分

开放评分和投票

撤掉评委身份，按活动规则开放评分，防重复提交，赛后转作品展示。

飞书

本地 + 飞书备份

本地数据库留底，附件本地备份，飞书 Base / 云盘作为协作和备份面板。

曝光

作品展示回看

按赛道筛选、排序、详情页展示附件与外链，保留赛后传播页面。
